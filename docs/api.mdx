## Blocks

- Blocks handle layout
- blocks can set basic colors for the entire block
- props
  - px/py
  - color/bg

```mdx
import { Centered } from 'mdx-blocks'
export default props =>
  <Centered
    {...props}
    color='white'
    bg='black'
  />
```

```mdx
import { Centered } from 'mdx-blocks'

<Centered
  color='white'
  bg='black'
  py={[ 5, 6 ]}
  css={{
    // ...overrides
  }}
/>
```
