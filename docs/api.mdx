## CSS-agnostic approach
- blocks *only* handle basic layout
- classNames are provided for all components
- other styles can be handled with plain CSS
- or by using something like emotion

```mdx
import { Centered } from 'mdx-blocks'
export default props =>
  <Centered
    {...props}
    color='white'
    bg='black'
  />
```

```mdx
import { Centered } from 'mdx-blocks'

<Centered
  color='white'
  bg='black'
  py={[ 5, 6 ]}
  css={{
    // ...overrides
  }}
/>
```
